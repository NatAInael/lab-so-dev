# PARTE A — Exploração da hierarquia
# Para cada diretório, registre o comando executado e uma linha
# explicando a função do diretório com suas próprias palavras.

/proc
 mostra processos que já foram criados e encerrados
/proc/cpuinfo
descreve sua cpu
  

/proc/meminfo
 da detalhes sobre a memória, como capacidade total, disponível e sem uso, ativa, etc.


/proc/uptime
 mostra o tempo em que o sistema foi ligado e o tempo que o sistema ficou ocioso


/bin
  lista todos  os executáveis no diretório, são os comandos básicos do sistema
/dev
  Cada item aqui  representa um dispositivo de hardware

/etc
  O diretório /etc contém os arquivos de configuração de praticamente todo programa instalado no sistema.
